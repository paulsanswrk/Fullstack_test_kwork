# Тестовое задание для фулстек разработчика

## Задачи

### Обязательные
#### Базовые
1. Поменять название текста ссылок на безликую надпись "Страница #" на подпись, соответствующую теме страницы (можно скопировать с h1).
2. Вынести стили в отдельный файл, при этом визуально стили на всех страницах должны остаться прежними.
3. Сделать отступы между ссылками одинаковыми на всех страницах.

#### JS
4. При клике на ссылки нужно, чтобы новая страница открывалась без перезагрузки и менялся URL. Для решения использовать библиотеку `assets/js/axnikitaJS.js`.

#### PHP
5. Сделать так, чтобы при открытии страницы с GET-параметром `simple=1` не подключались стили и скрипты.

### Дополнительно
6. Добавить страницу по шаблону остальных, контент сгенерировать нейронкой. Все пункты выше должны так-же работать с новой страницей.
7. Сделать так что-бы файлы с раширением .php открывались просто по имени без расширения.
